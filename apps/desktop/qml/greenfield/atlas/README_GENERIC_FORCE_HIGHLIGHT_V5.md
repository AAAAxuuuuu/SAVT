# SAVT Edge Readable Graph v5

Version tag: `readable-generic-force-highlight-v5-arrow-visible-spacing`

Changes from v4:

1. Arrow heads are medium sized and wider, so they read clearly as arrows instead of short line caps.
2. Arrow clearance from node cards is increased to avoid heads being swallowed by card edges.
3. Node layout spacing is expanded: larger rings, stronger collision gaps, wider isolated sidebar gap.
4. The logic remains generic: default focus is selected by graph degree, bidirectional pairs are merged, isolated nodes are placed aside.
