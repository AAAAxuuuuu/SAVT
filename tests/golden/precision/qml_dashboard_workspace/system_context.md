# 系统上下文 (L1)

## 发起入口
- **App** (procedure_centric)：作为项目的启动入口，负责初始化运行环境并将控制权交给核心模块。

## 核心能力
- **Backend Controller** (interaction)：承担项目中某项核心业务能力，参与主要的用户可见工作流。
- **Ui** (🎨 界面展示 (UI))：负责渲染界面和处理用户的点击/滑动等操作。

## 后台支撑
- **Backend Service** (interaction)：提供底层支撑服务，保持核心能力流程的稳定性和可复用性。
- **Data** (data)：负责持久化存储配置、资源或项目运行数据，供其他模块读取和写入。
